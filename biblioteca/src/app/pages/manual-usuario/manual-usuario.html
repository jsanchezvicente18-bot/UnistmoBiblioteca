📖 Manual de Usuario

Bienvenido a la Biblioteca Digital UNISTMO. Este manual explica cómo utilizar las funciones disponibles desde la pantalla de inicio de sesión.

📝 Crear una cuenta

  1. Haz clic en Crear cuenta.
  2. Completa los datos solicitados (nombre, matrícula, correo, teléfono y contraseña).
  3. La contraseña debe contener:
    • Mínimo 8 caracteres.
    • Una letra mayúscula.
    • Un número.
    • Un símbolo especial.
  4. Presiona Registrar.

🔑 Iniciar sesión

  1. Escribe tu matrícula o correo electrónico.
  2. Ingresa tu contraseña.
  3. Presiona Iniciar sesión.

🔐 Recuperar contraseña

  1. Haz clic en Recuperar contraseña.
  2. Ingresa el correo con el que te registraste.
  3. Escribe una nueva contraseña.
  4. Confirma la contraseña.
  5. Serás redirigido nuevamente al inicio de sesión.

📧 Contactar al administrador

Si tienes problemas para acceder al sistema, utiliza la opción Contactar administrador para consultar los datos de contacto.

🚨 Reportar un problema

Si encuentras un error en el sistema, selecciona Reportar error, completa el formulario y envía la información al administrador.

Importante: Una vez que inicies sesión, tendrás acceso únicamente a las funciones correspondientes a tu tipo de usuario (estudiante o profesor).
